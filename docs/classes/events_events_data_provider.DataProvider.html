Class DataProvider

The DataProvider facilitates interactions with the Hourglass RFQ system /data namespace.

The /data namespaces enables clients to query information about the Hourglass RFQ system. The current set of supported actions is:

  • Query markets (see DataProvider.getMarkets)

This class extends an event emitter and proxies events from the underlying websocket to itself so that SDK consumers can listen for events without managing the websocket.

Example

const dataProvider = new DataProvider();

Actions

getMarkets

  • getMarkets(): void

  • Query the list of supported markets.

    • This method triggers the emission of a 'message' event to the server.
    • The listener for event DataMethod.hg_getMarkets will receive the response.
    • If successful, the type of the response object will be PayloadHgGetMarkets.

    Returns void

    Example

     dataProvider.getMarkets();
     dataProvider.on(DataMethod.hg_getMarkets, (data: PayloadHgGetMarkets, error) => {
       if (error) {
         console.error(`Error querying markets: ${error}`);
         return;
       }
       console.log(`Markets: ${markets}`);
     });
